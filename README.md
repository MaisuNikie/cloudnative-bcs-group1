# cloudnative-bcs-group1
You will need `docker`, `docker-compose`, and [Git for Windows](https://gitforwindows.org/) to run this project.

Use the `start.sh` script to start the project, it should take care of everything automaticly

## Starting project
## Linux & Mac
#### Start Project
```bash
./start.sh
```
### Windows
#### Open `Git for Windows`
#### Navigate to project directory
```bash
cd project/directory/
```
#### Start Project
```bash
bash start.sh
```
